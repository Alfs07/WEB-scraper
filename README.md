# WEB-scraper
pengumpulan data aplikasi di google play store untuk analisis statistik deskirptif, dimana akan menghitung sentimen ulasan. 
kode ini akan menghitung 
-mean                      -varians            -Q3
-median                    -max                -IQR
-modus                     -min                -jumlah data 
-standar deviasi           -Q1

lalu kode ini akan menghitung distribusi frekuensinya, dengan kategori:
bintang 1-2 = 'negatif'
bintang 3 = 'netral'
bintang 4-5 = 'positif'
setelah menghitung semua itu, kode ini akan menghitung distribusi sentimen ulasan apk dan akan di persenkan.

selanjutnya adalah visualsasi distribusi frekuensi, tren apk perhari (rata-rata sentimen perhari), pie-chart (distribusi sentiment)  dan boxplot.

selanjutnya akan mengexport file ulasan sebagai csv (APK_ulasan.csv), dan ringkasan statistik (APK_statistik.xlsx)
